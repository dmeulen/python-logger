# python-logger
A logger implementation in python

  usage: pylogger.py [-h] [--facility FACILITY] [--tag TAG] [--cee]

  optional arguments:
    -h, --help            show this help message and exit
    --facility FACILITY, -f FACILITY
                          Syslog facility
    --tag TAG, -t TAG     Syslog tag
    --cee, -c             Add CEE cookie
